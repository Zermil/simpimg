# Simpimg

Rudimentary image viewer made with [OpenGL](https://www.opengl.org/)

![example](./example.png)

## Build

Build for release:

```console
> build_release.bat
```

Build for debug:

```console
> build.bat
```

Remember to change `MSVC_PATH` variable inside the build scripts, otherwise it won't be able to execute `cl.exe`
